# Dotfiles

Personal configuration files for various tools and applications.

## Contents

- **alacritty**: Terminal emulator configuration with themes
- **ghostty**: Configuration for the Ghostty terminal
- **iterm2**: iTerm2 terminal configuration with color schemes
- **nvim**: Neovim configuration with plugins and customizations
- **yazi**: Configuration for Yazi file manager

## Installation

Run the included install script to set up the configurations:

```bash
./install.sh
```

## Terminal Themes

This repository includes numerous color schemes for terminal emulators:
- Alacritty themes in `alacritty/themes/`
- iTerm2 color schemes in `iterm2/iterm2-themes/`
- Ghostty themes in `ghostty/themes/`
